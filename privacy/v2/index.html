Privacy Policy for SixFour Chess

Last Updated: February 15, 2026

Publisher: ADOUANI SAS

Application: SixFour - Chess Game (iOS)

Select Language: English Français Italiano

Overview

SixFour is a chess application for iOS. It offers both offline and online game modes. We are committed to protecting your privacy and being transparent about our data practices in compliance with GDPR and CNIL regulations.

Data Controller

ADOUANI SAS

Email: adouani.sas@gmail.com

Website: github.com/adouanisas/six-four-chess-site

Data Collection

Offline Mode (default) NO DATA COLLECTED

In offline mode, SixFour does NOT collect any personal data.

Online Mode (optional with your consent) REQUIRES CONSENT

If you choose to enable online multiplayer mode, we use Firebase (Google) to:

  • Create online games (anonymous game ID)
  • Synchronize game state between players
  • Manage matchmaking

Data collected in online mode:

  • Game ID (randomly generated, anonymous)
  • Moves played during the game
  • Current game state
  • Action timestamps
  • Display name (player-chosen username)
  • Device installation identifier (anonymous)
  • Game statistics (games played, games won)

This data:

  • Uses anonymous authentication (no email or password required)
  • Is stored on Firebase for the duration of use
  • Is NEVER used for advertising purposes

Data NOT Collected

We do NOT collect:

  • Real name, surname, or legal identity
  • Email address
  • GPS location
  • Contacts
  • Photos
  • Sensitive data

Local Data Storage

All game data is stored locally on your device only:

  • Your game history
  • Your settings and preferences
  • Your saved games
  • Your game statistics

This data:

  • Remains on your device at all times
  • Is never transmitted to any server
  • Is never shared with anyone
  • Can be deleted by uninstalling the app

Permissions

SixFour only requires the following permissions:

  • Network access: Only if you enable online multiplayer mode (optional)
  • No location access
  • No camera access
  • No microphone access
  • No photo library access
  • No contacts access

Third-Party Services

Firebase (Google Cloud)

SixFour uses the following Firebase services (Google), all subject to user consent:

Firebase Realtime Database (online multiplayer)

Used only if you enable online multiplayer mode:

  • Hosting online games
  • Real-time game state synchronization
  • Anonymous authentication (no email or password required)

Firebase Analytics (performance analytics) REQUIRES CONSENT

Only activated with your explicit consent. Collects anonymous usage data:

  • Game events (game started, finished, moves count)
  • No advertising identifier (IDFA) is collected

Firebase Crashlytics (crash reporting) REQUIRES CONSENT

Only activated with your explicit consent. Collects crash data to help us fix bugs:

  • Crash reports and stack traces
  • Device model and OS version
  • No personally identifiable information

Data processor: Google LLC (Firebase), acting as data processor under GDPR Article 28.

Data Processing Agreement: Covered by Firebase Data Processing Terms.

Firebase policy: firebase.google.com/support/privacy

No Advertising Services

SixFour does NOT use:

  • Any advertising platforms
  • Any advertising identifiers (IDFA)
  • Any social media SDKs

Children's Privacy

SixFour is safe for users of all ages, including children. Since we do not collect identifying personal data, there are minimal privacy concerns for children using the app in offline mode.

Data Security

Your game data is protected by:

  • iOS Sandbox security (app isolation)
  • iOS data encryption at rest
  • HTTPS encryption for online data transmission
  • Firebase security rules

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date above.

Your Rights (GDPR)

In accordance with GDPR, you have the following rights:

Right of Access

You can request what data we hold about you. For online mode, you can view your active games in the app.

Right to Rectification

You can change your display name at any time within the app. For other corrections, contact us.

Right to Erasure

  • Offline mode: Uninstall the app to delete all local data
  • Online mode: Contact us to request deletion of your player profile and game data
  • Explicit request: Contact us at adouani.sas@gmail.com to delete specific data

Right to Data Portability

You can export your local game data via the app's export features.

Right to Object

You can refuse to use online multiplayer mode at any time via consent settings.

Exercising Your Rights

To exercise these rights, contact us at: adouani.sas@gmail.com

Response time: Maximum 30 days

Consent Management (Cookies & Trackers)

SixFour uses a CNIL-compliant Consent Management Platform (CMP).

Consent Categories

1. Strictly Necessary ALWAYS ACTIVE

  • Basic app functionality
  • Local game saving
  • User settings

2. Functional OPTIONAL - REQUIRES CONSENT

  • Online multiplayer mode
  • Game synchronization via Firebase

3. Performance & Analytics OPTIONAL - REQUIRES CONSENT

  • Firebase Analytics (anonymous usage statistics)
  • Firebase Crashlytics (crash reporting)

Managing Your Preferences

You can change your consents at any time in Settings → Cookies and Consent.

Your choices are stored locally and respected immediately.

Legal Basis for Processing

  • Offline mode: Legitimate interest (Article 6(1)(f) GDPR)
  • Online mode: Explicit consent (Article 6(1)(a) GDPR)
  • Analytics & Crashlytics: Explicit consent (Article 6(1)(a) GDPR)
